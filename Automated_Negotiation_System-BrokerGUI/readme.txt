For our program, following libraries need to be added to the project:

- javafx library: e(fx)clipse version 3.0.0.201705220750
- commons-io-2.6
- jackson-annotation-2.9.9
- jackson-core-2.9.9
- jackson-databinf-2-9-9
- jade

All libraries are stored under the folder lib (except the javafx library).
